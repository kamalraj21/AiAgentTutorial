# Pocket AI Agent

Build your own mobile AI agent in 5 minutes. No coding required.

## Quick Start
1. Get API key: https://console.groq.com
2. Download `my-agent.html`
3. Open on phone → Add to Home Screen
4. Paste key + customize instructions

- Marketing Manager (Content Repurposer)
